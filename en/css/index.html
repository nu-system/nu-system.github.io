# What?

css-[component] Mainly responsible for the UI and Skin Library in nu-system„ÄÇ

# How to start

$ yarn global add @_nu/cli
$ nu-cli # and select css

It will create a master component for you.

After finish your can go nu-system github and create new repository.

And push your component.

# Style Guild

  • Prefixed with nu_, like .nu_btnÔºõ
  • Status recommended to use native or group selectors with prefix _ like .nu_btn:disabled, .nu_btn._loadingÔºõ

# Bad ‚ùå

<style>
.nu_btn{}
.nu_btn_disabled{}
.nu_btn_loading{}
</style>
<button type="button" class="nu_btn nu_btn_disabled nu_btn_loading">
    hello world!
</button>

# Good ‚úÖ

<style>
.nu_btn{}
.nu_btn:disabled{}
.nu_btn._loading{}
</style>
<button type="button" class="nu_btn _loading" disabled>
    hello world!
</button>
